# Hand Gesture Recognition + Gesture Battle Game

This project implements a **real-time hand gesture recognition system** using **MobileNetV3-Small** and a **gesture-controlled Pygame mini-game**.  
The system supports six gestures (`up`, `down`, `left`, `right`, `stop`, `zero`) and includes:

- A webcam dataset collection tool  
- A retraining script with per-class augmentation  
- A Kaggle RGB training script  
- A simple gesture test game  
- A full gesture battle game (player vs. enemy HP system)

---

## Project Structure

| File | Description |
|------|-------------|
| `collect_gestures.py` | Webcam tool for collecting your own gesture dataset. |
| `train.py` | Trains MobileNetV3 on the Kaggle RGB dataset. |
| `retrain_model.py` | Retrains the model using your **self-collected dataset**, with per-class augmentation. Generates evaluation matrix + accuracy curve. |
| `test_game.py` | Simple demo: move a green cube using gesture predictions. |
| `game_noimg.py` | Final mini-game (no images): player (blue circle) vs enemy (red square), gesture-based attacks. |
| `best_model_perclass.pth` | Saved model after retraining (generated by scripts). |
| `dataset_retrain/` | Your self-captured gesture dataset organized into folders. |

---

## 1. Collecting Your Own Dataset

Run:

```bash
python collect_gestures.py

Controls
| Key | Gesture  |
|-----|----------|
| u   | up       |
| d   | down     |
| l   | left     |
| r   | right    |
| s   | stop     |
| z   | zero     |

```
Images are saved automatically into:
dataset_retrain/<gesture_name>/

## 2. Retraining the Gesture Model (Your Own Dataset)

Run:

```bash
python retrain_model.py
```

This script:

- Applies per-class data augmentation

- Splits into train / val / test

- Saves the best model to best_model_perclass.pth

- Generates an evaluation metrics

- Generates a training/validation accuracy curve

## 3. Simple Gesture Test Game

Run:

```bash 
python test_game.py
```
This mini demo will test the new model:

- Opens webcam when you press SPACE

- Predicts gesture from a single frame

- Moves a green cube based on:

    - up / down / left / right

    - zero → reset cube

    - stop → do nothing

## 4. Gesture Battle Game

Run:
```bash
python game_noimg.py
```

Gameplay:

- A sequence of 4 gestures (up/down/left/right) is generated

- Press SPACE to open webcam → pose → press SPACE again to submit

- Correct gestures progress sequence

- Completing all 4 reduces enemy HP by 20%

- Wrong gestures don’t punish you — you retry the same gesture

- Timer counts down from 120 seconds

- First HP bar to hit 0 loses

## 5. Results

- Kaggle dataset: near 100% training accuracy with stable convergence
- Personalized dataset:
  - Validation Accuracy: ~100%
  - Macro F1 Score: 0.98–1.0
  - Perfect confusion matrix (no misclassification across all gesture classes)

These results show that the model adapts effectively to user-specific data and achieves strong classification performance.

## 6. Real-Time Performance

Although offline performance is near-perfect, real-time gameplay reveals practical challenges:

- Sensitivity to lighting conditions
- Motion blur during gesture execution
- Inconsistent hand positioning within ROI
- Occasional misclassification in live webcam input

To address this, the game uses a **prompt-based interaction design** (capture-on-demand) instead of continuous gesture tracking, improving reliability during gameplay.
